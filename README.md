# 🍽️ Oltremare — Система управления рестораном

> Десктопное приложение для автоматизации работы ресторана «Oltremare».  
> Разработано в рамках учебной практики **УП 02.01** по специальности **09.02.07**.

---

## 📋 Описание

Система автоматизирует ключевые бизнес-процессы ресторана:

- **Бронирование столиков** — с проверкой вместимости и конфликтов по времени
- **Управление заказами** — полный цикл от составления до выдачи клиенту
- **Контроль кухни** — очередь заказов и управление статусами
- **Смены официантов** — формирование графика, открытие/закрытие смены
- **Склад кухни** — остатки блюд с автоматическим уменьшением при заказе
- **Акции и скидки** — по блюдам и категориям, с датой действия
- **Счета и чеки** — формирование, оплата, уникальный номер чека
- **Статистика** — продажи, брони, работа официантов, занятость по часам

## 🚀 Установка и запуск

### 1. Требования

- Python 3.10 или новее
- MySQL 8.0 или MariaDB 10.6+
- pip

### 2. Клонировать репозиторий

```bash
git clone https://github.com/<ваш_логин>/oltremare.git
cd oltremare
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Настроить подключение к БД

Откройте файл `src/database/db.py` и измените параметры:

```python
DB_CONFIG = {
    'host': 'localhost',
    'port': 3306,
    'database': 'oltremare_db',
    'user': 'root',       # ← ваш логин MySQL
    'password': '',       # ← ваш пароль MySQL
}
```

> ⚠️ База данных `oltremare_db` создаётся **автоматически** при первом запуске.

### 5. Запустить приложение

```bash
python main.py
```

## 🗃️ Структура базы данных

```
oltremare_db
├── roles               — роли пользователей
├── users               — пользователи системы
├── tables              — столики ресторана
├── shifts              — смены официантов
├── shift_tables        — столики в смене
├── reservations        — брони
├── reservation_tables  — столики в брони
├── dish_categories     — категории блюд
├── dishes              — блюда (меню + склад)
├── promotions          — акции и скидки
├── orders              — заказы
├── order_items         — позиции заказа
├── bills               — счета
└── receipts            — чеки
```

**Нормализация:** все таблицы соответствуют **3НФ**.  
**Движок:** `InnoDB`, поддержка транзакций и внешних ключей.
